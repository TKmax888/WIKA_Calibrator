Achtung es gibt zwei Versionen. 
Die erste heißt nur "WIKA" und wird nur auf der Komandozeile ausgeführt. Der einzige Parameter ist der COM Port. 
Es schaltet zwischen den SP Einstellungen um, wenn die Bedingungen erfüllt sind. Bedingungen bedeutet stabile Temperatur für eine bestimmten Zeitraum. Voreingestellt ist +-0,3° für 10 sekunden.
Die Soll und Istwerte werden angezeigt und auch in einer Datei gespeichert.
Die zweite Version heißt "WIKA_GUI" und wie der Name vermuten lässt gibt es hier noch eine Benutzeroberfläche.
Der COM-Port und die Betriebsart können auf der Oberfläche ausgewählt werden. 
Angezeigt wird der aktuelle Sollpunkt, sowie die aktuelle Soll und Isttemperatur.
Die Bedingungen zur Umschaltung zum nächsten Sollpunkt kann ausgewählt werden. Also die Stabilitätsdauer und die Temperaturtoleranz.
Welche Sollpunkte überhaupt angefahren werden kann ebenfalls ausgewählt werden.
Ganz unten wird die Aufzeichnung der Messwerte gestartet. Wenn kein Sollpunkt ausgewählt wird, dann wird ohne Zeitlimit aufgezeichnet.
